# Guion de Video Promocional: Utileria.js 🎬
**Duración estimada:** 60 segundos
**Objetivo:** Vender la utilidad y simplicidad de la librería para validaciones en desarrollo web.

---

| Tiempo | Acción en Pantalla (Video) | Narración de Voz (Audio) |
| :--- | :--- | :--- |
| **0:00 - 0:10** | *Comienza con una toma del código HTML/JS de validación largo y desordenado en VS Code. Transición rápida al formulario web vacío.* | ¿Cansado de escribir decenas de líneas de código repetitivo solo para validar un simple formulario? ¿Harto de lidiar con expresiones regulares confusas una y otra vez? |
| **0:10 - 0:18** | *Muestra el archivo `index.html` e introduce la etiqueta `<script src="js/utileria.js"></script>` resaltándola con el cursor.* | Te presento **Utileria.js**. La librería ligera en JavaScript diseñada para resolver tus dolores de cabeza. Instalarla es tan fácil como agregar una etiqueta `<script>` en tu HTML. |
| **0:18 - 0:35** | *Enfoca el formulario principal. Llena campos incorrectos (ej. número telefónico larguísimo, código postal con letras) y haz clic en "Validar". Muestra los textos de error en rojo y la consola con errores.* | Con funciones listas para usar, valida correos, contraseñas seguras, códigos postales y fechas en tiempo récord. Mira qué pasa si introduces datos inválidos: ¡La librería bloquea el envío al instante y te lo muestra tanto en pantalla como en la consola! |
| **0:35 - 0:50** | *Corrige los campos rápidamente (ej. "Jose Daniel", número corto, código postal de 5 dígitos). | **0:50 - 1:00** | *Termina mostrando el repositorio de GitHub con el README y el enlace a GitHub Pages en pantalla.* | Ahorra tiempo, mantén tu código limpio y asegura tus datos hoy mismo. ¡Lleva tus formularios al siguiente nivel con **Utileria.js**! Pruébalo ahora. |

---

### 💡 Consejos para la grabación:
1. **Tono de voz:** Entusiasta, dinámico y seguro (estilo presentación de producto tecnológico o TikTok/Reel informativo).
2. **Edición:** Haz cortes rápidos en los momentos de transición para mantener el dinamismo y no sobrepasar el límite de 60 segundos.
3. **Calidad de audio:** Graba en un espacio silencioso utilizando tus auriculares o un micrófono dedicado para que tu voz se escuche clara.
